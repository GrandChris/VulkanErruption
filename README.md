# VulkanErruption

Vulkan GLFW application for rendering constantly changing particles.

## Goals

+ Draw constantly changing particles
+ Easy to include in another application
+ Easy to use

## Advanced Goals

+ Add VR support

## Extended Advanced Goals

+ Add support for Cuda

## Dependencies

+ GLFW 3.3
+ Vulkan 1.1.114.0
+ GLM 0.9.9.5
